macOS — Terminal

1) Generate an SSH key

ssh-keygen -t ed25519 -C "your.email@campus.lmu.de"

Press Enter to accept the default path ~/.ssh/id_ed25519. A passphrase is optional.

2) Edit your SSH config

nano ~/.ssh/config

Paste the snippet from the widget above (the one with your ephysNN filled in). Save with Ctrl+O, Enter, then Ctrl+X.

3) First login (with password)

ssh gamma3

Use the temporary password from the course chat. Type exit to log back out.

4) Install your public key

ssh-copy-id gamma3

5) Verify passwordless login

ssh gamma3   # should connect without asking for a password

Linux — Terminal

1) Generate an SSH key

ssh-keygen -t ed25519 -C "your.email@campus.lmu.de"

2) Edit your SSH config

nano ~/.ssh/config

Paste the snippet from the widget above. Make sure ~/.ssh has mode 700, otherwise ssh will refuse to use it:

chmod 700 ~/.ssh
chmod 600 ~/.ssh/config

3) First login (with password)

ssh gamma3

4) Install your public key

ssh-copy-id gamma3

If ssh-copy-id is missing on your distro, fall back to the manual steps shown in the Windows tab.

5) Verify passwordless login

ssh gamma3

Windows — PowerShell or Git Bash

1) Generate an SSH key

ssh-keygen -t ed25519 -C "your.email@campus.lmu.de"

Default location is C:\Users\<you>\.ssh\id_ed25519.

2) Edit your SSH config

Create (or edit) C:\Users\<you>\.ssh\config in Notepad and paste the snippet from the widget above.

3) First login (with password)

ssh gamma3

4) Install your public key

Show your public key:

type $env:USERPROFILE\.ssh\id_ed25519.pub

Copy the whole line (starts with ssh-ed25519), then on the server:

ssh gamma3
mkdir -p ~/.ssh && chmod 700 ~/.ssh
notepad ~/.ssh/authorized_keys     # or: nano ~/.ssh/authorized_keys
# paste your key, save, exit
chmod 600 ~/.ssh/authorized_keys

Prefer PuTTY? Generate an Ed25519 key with PuTTYgen, save the .ppk, and paste the public-key text into authorized_keys. In PuTTY, save a session named gamma3 with HostName 10.153.170.43 and your username.

5) Verify passwordless login

ssh gamma3
